## 一、要部署的兩個檔案

| 檔案 | 平台 | 步驟 |
|---|---|---|
| **Code.gs** | Google Apps Script | 貼回編輯器 → 部署 → 管理部署作業 → ✏️編輯 → 版本選「**新版本**」→ 部署 |
| **index.html** | GitHub Pages | 重新上傳(蓋掉舊的)→ 開啟時網址換新的 `?v=` 數字 |

> ⚠️ **關鍵**:Code.gs 只按 Ctrl+S 存檔**不會更新 `/exec`**,一定要「部署新版本」;否則會出現「unknown action」。
> ⚠️ index.html 一定要**換 `?v=` 數字**避開手機快取,否則載到舊版。

## 二、部署後一次性設定(改善冷啟動最有感)

### 1. 保溫觸發器(必做)
Apps Script 後台 → 函式下拉選 `setupKeepWarm` → 執行(每 5 分鐘保溫)。

### 2. 外部保溫(強烈建議,對「開 App 很慢」最有效)
到 **cron-job.org**(免費)新增 job:
- URL:`你的GAS網址/exec?action=ping`
- 間隔:每 5 分鐘

### 3. 其他一次性
- Config 若有 `eventGraceDays = 5` → 改成 **3**(或刪該列用預設)。
- 若要備份:執行 `setupBackupTrigger` 一次。
- 幹部欄位:管理員「其他管理 → 球隊資料」填齊(會長/副會長/財務長/總幹事)。
