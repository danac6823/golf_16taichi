# 更新摘要(Changelog)

> 意象太極高爾夫球隊 · 例賽管理系統
> 本批更新整理(2026/6)。給總幹事日後回顧 / 交接用。

---

## 🆕 新功能

### 報名 / 群組互動
- **分享到群組**:系統內一顆「📤 分享到群組」,點開可選兩種卡片送到 LINE 群組,卡片都附「我要報名/請假」按鈕連回系統。
  - **🏌️ 例賽卡**:隊名・第X屆、日期 / 球場 / 開球 / 截止。
  - **📋 報名狀況卡**:已報名人數 / 名額、日期 / 球場 / 截止、目前報名名單(催報名用)。
  - ⚠️ 只能在 LINE App 內用;OpenChat(公開聊天)不支援。一次性設定見下方。

### 公告 / 匯款
- **三塊公告區**:📣 最新消息/外地賽(首頁上方常駐)、📌 本場好康(報名卡下方、換月自動清空)、💳 匯款資訊(只對未繳會員在「我的會費」顯示、一鍵複製帳號)。都在管理員「公告 · 本場好康 · 匯款資訊」面板設定,網址自動變連結。

### 會員身分 / 名單
- **LINE 名稱(唯讀)+ 姓名分開**:初次設定畫面把「LINE 名稱」設成唯讀只供核對,另開「姓名(中文全名)」欄請填本名,避免有人用 LINE 暱稱誤綁。LINE 名稱會存檔。
- **綁定名單顯示更多**:管理員「綁定狀態」可看到每人的 LINE 名稱、電話,並標示「· 新會員」。
- **代會員報名**:幫還沒綁定 LINE 的會員登記出席(身分算會員、成績照算、會進分組),他日後綁定自動接上;出席名單匯出會在其後標「(代)」。同時修正:未綁定會員的性別/出生年改從「會員名單」讀,淨桿平手比年齡與二男二女分組更準。
- **同步名單姓名**:若有人誤用暱稱綁定,改好「會員名單」姓名後,按「🔄 同步出席名單姓名」一鍵更新出席名單(依 LINE 帳號對應,差點也重抓,不必大家重報)。

### 會費
- **會費調整**:女會員 **10000**、男會員 **11000**(來賓無年費);改成可由 Config `feeFemale`/`feeMale` 調整。

### 差點 / 計分
- **Birdie 欄自動技術獎**:成績輸入表新增 Birdie 計數欄,送出時自動列入 BIRDIE獎(每人最多 2 條、與手動列合併去重),固定預帶列不再含 BIRDIE。
- **差點三層退路**:顯示/計算差點時 目前差點 → 期初差點 → 會員名單「年初差點」依序退路。**還沒比賽、還沒匯入 Handicaps 時也看得到每人的年初差點**(解決「期初有值卻顯示—」)。
- **新會員規則**:在「會員名單」加「會員身份」欄標「新會員」者,**第一場只記總桿不排名**,**第二場**用前兩場平均自動算差點=`round(((第1場+第2場)/2 − 標準桿)×0.9)`,當即算淨桿並進排名;之後照一般會員跑。倍率用 Config 的 `newHcpFactor`(預設 0.9)可調。
- **差點上限 36**:所有系統算出的差點(年初重設、新會員建立、賽後扣桿)一律夾在 `0~maxHcp`(預設 36,Config 可調)。手動填的年初差點/匯入差點不強制夾。

### 首頁改版(方塊導向)
- **首屏合併排版**:標題改「日期 · 球場短名」;資訊欄保留球場全名/開球,費用｜名額 2 欄對齊,截止列自動顯示「剩 N 天/今天截止」;統計(參加/請假/尚未登記)併入事件卡底;「你」身分列與出席操作合併成一張卡。本場好康維持獨立條。
- **請假/取消按鈕加小字**:改為請假「不打了,先告知」、取消參加「收回登記,回到未登記」,一眼分辨差異。
- **分享改方塊**:「📤 分享群組」成為功能方塊(管理員限定),點開選例賽卡/報名狀況。
- **Birdie 欄自動技術獎**:成績表新增 Birdie 計數欄,自動列入 BIRDIE獎(每人最多 2 條)。
- **讀取加速**:快照秒開(上次畫面即開、背景刷新)+ 繳費狀態快取。
- **混合式首頁**:首屏報名核心不變;中後段長捲動改為 3 欄**功能方塊**(出席名單/分組/成績/年度總表/賽程/我的統計/會費/來賓/消息/管理),點開為全螢幕頁,含狀態小字與紅點。幫來賓報名由「你的出席」移入方塊。
- **換場也清分組**:換到新場次時,除名單/本場好康外,一併自動清空上一場分組,避免新場次顯示舊組。

### 換場 / 報名
- **換場自動化**:目前場次改由行事曆日期自動判斷,賽後寬限由 5 天改 **3 天**;比賽日過 3 天就自動換到下一場。
- **換場自動清空 + 分月留存**:換到新場次時,系統自動把上一場報名**封存到「報名紀錄」分頁**(每列標場次日期)再清空名單,開始新一輪報名。不必再手動清空;歷史每月報名查得到。

### 出席名單 / 賽程
- **匯出出席名單(傳球場)**:純文字含 隊名・第X屆、日期 / 球場 / 開球、編號名單,結尾帶「承辦:總幹事 … 電話」,可直接貼 LINE。
- **整年度賽程改滿版**:全螢幕檢視,清單可順暢捲動。

## ⚡ 效能 / 穩定

- **登入加速三層**:① 保溫觸發器(降低冷啟動)② 前端 localStorage 並行暖機 ③ **CacheService 跨請求暖快取**(賽前大量登入時,第一位讀過後第二位以後幾乎不必再讀表)。
- **暖快取即時失效**:App 內任何操作(報名 / 改設定 / 輸入成績…)會立即更新;只有**直接手動改試算表**需等 10 分鐘或按「🔄 清除快取」。
- **清除快取按鈕**:手動改表後一鍵讓 App 讀到最新。
- **批次寫入 / 延後載入 / 搶報名鎖**:匯入、成績、收款等一次讀一次寫;管理員重資料用到才抓。

## ✅ 一次性設定待辦(只做一次)

> 這些不做的話對應功能不會生效。

- [ ] **發布 LINE Login 頻道**(Developing → Published)→ 否則一般會員看到「400 Bad Request… developing status」無法登入。
- [ ] **啟用 shareTargetPicker**(LINE Developers → LIFF 分頁,同意條款)→ 否則「分享到群組」按下去會提示尚未啟用。
- [ ] Apps Script 執行一次 **`setupKeepWarm`**(保溫加速登入)。
- [ ] Apps Script 執行一次 **`setupBackupTrigger`**(例賽隔天自動備份試算表,保留 30 份)。

| 檔案 | 動作 |
|---|---|
| `Code.gs` | 貼回 Apps Script → 部署 → **「新版本」**(只存檔不會生效) |
